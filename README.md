# Sports Manager
This is our term project called Sports Manager. The initial version includes the core foundation classes and a functional football simulation.

## Features
- Core foundation classes fully implemented
- Football-specific logic and simulation integrated
- Includes a single-match demo
- JUnit tests provided for core logic

## How to Run
- **To run the live demo:** Execute `src/main/java/sportsmanager/Main.java` in your IDE or use the command `mvn exec:java`
- **To run the tests:** Use your IDE's test runner or use the command `mvn test`
